# README

初于个人练习的考虑，本仓库实现了一些数字几何处理的算法，涵盖网格简化、变形、优化、去噪、参数化等。

## 网格参数化

网格参数化常用在渲染中，例如纹理，法向渲染等。数学描述问题的一般形式为：\\(f:\mathbb{R}^3\rightarrow\mathbb{R}^2\\)
